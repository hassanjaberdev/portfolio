## ⚙️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/hassanjaberdev/portfolio.git
cd portfolio
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm run dev
```

---

## 🏗️ Build for production

```
npm run build
```
